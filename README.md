# CardLumus 🎴
Нативное iOS приложение для изучения слов с использованием UIKit.

## 🛠 Технологии
- **Language:** Swift
- **UI Framework:** UIKit
- **Architecture:** MVC
- **Layout:** Compositional Layout, AutoLayout
- **Storage:** UserDefaults + Codable

## 🚀 Функционал
- Создание и удаление категорий.
- Интерактивные карточки с жестами (свайпы).
- Динамический неоновый фон, реагирующий на действия пользователя.
- Сохранение прогресса обучения.
